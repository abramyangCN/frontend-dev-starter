# Frontend Development Starter

A frontend development starter powered by Gulp, Alpinejs, Tailwindcss, Babel, Webpack, SASS & BrowserSync.

## Requirements

- [Nodejs](https://nodejs.org/)
- [NPM](https://www.npmjs.com/get-npm) or [Yarn](https://yarnpkg.com/) or other node package manager

## Features

- HTML Layouting
- Module Bundler
- Live reloading
- Image Optimization
- Others

## Getting Started

- Clone this repo with `git clone https://github.com/ariakm25/frontend-development-starter` or download the zip.
- Run `npm install`
- Run `npm run watch` to start development mode
- Your browser should open at http://localhost:3000
- Edit your code inside of the src folder.
- Run `npm run build` to compile as production mode. Your compiled and minified minified css, html, and javascript files will be created and updated in `dist` folder.
